What is qSim?

qSim is a QoS simulator that you can use to measure, how different QoS mechanizms work.
The simulator is supposed to be as realistic as possible, so here is a list of parameters you can specify during simulation:
  • link bitrate and length [meters]
  • link reliability
  • simulation of input/output queues
  • simulation of RX/TX buffers
  • size of routers' buffers/queues
  • and many more

Why qSim?

There are plenty of network simulators out there (NS2, Packet Tracer,...), but you really need to know these programs well, before you can start using them. Sometimes you even need to me your own programs/scripts to start a simple simulation. This is just wrong!
qSim was designed to make network simulation easy and fast. And not only that - existing network simulators cannot visualise all the data you need to optimise your network in terms of QoS. There are no charts that can tell you, how packet delay changed, there are no statistical data about buffer usage, etc. So these are the reasons, why was qSim created.
