Controls

Basic controls

The main controls are located at the top toolbar, which consists of buttons for:
  • run , stop and pause simulation
  • configuring simulation rules
  • show vertex statistics
  • simulation speed controls and
  • copy and paste vertices
  • buttons for project managing (creating new project, new file, open project,...)

Topology controls

  • holding right mouse button allows you to move around network topology
  • scrolling by mouse wheel zooms in/out network topology
