Computer

How to create computer

  1. select Palette
  2. choose Computer
  3. click anywhere on the topology - here will be the new computer placed
  4. configure your computer
  5. press OK button

Computer hardware configuration

  • maximum number of packets that can be processed in paralel
  • TX and RX buffers' size (number of fragments)
  • input queue size (number of packets)
  • output queue count and size (number of packets)
  • packet processing delay (depends on currenct number of processing packets - more packets are being processed, longer the delay)
  • TCP retransmission delay - number of msecs, before supposedly lost packet will be retransmitted
