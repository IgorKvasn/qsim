Edge

How to create edge

  1. select Palette
  2. choose desired edge type
  3. select first vertex (vertex will gain orange border)
  4. select second vertex (the order of the selected vertices is not important - edges are bidirectional)
  5. configure edge
  6. press OK button

Edge configuration

  • bitrate [MBit/s]
  • length [m]
  • maximum transmission unit [B]
  • error rate
    • bit error rate
    • packet error rate
