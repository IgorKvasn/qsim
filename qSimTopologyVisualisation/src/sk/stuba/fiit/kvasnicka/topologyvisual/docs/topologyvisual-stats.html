Statistics

Vertex statistics

Vertex statistics allows you to view load on selected vertices (routers, switches or computers).
Following statistical data can be observed:
  • RX and TX - overall or per interface
  • input queues
  • output queues - overall or per queue
  • number of packets currently in processing
Each of these items can be observed in textual form in table or drawn onto chart.
All values are updated automatically in real-time.

How to select vertex statistics

Vertex statistics can be opened by clicking on upper toolbar's "chart" icon . Panel in bottom will be opened.

Simulation rule statistics

Simulation rule statistics can visualise packet delay variation throughout the time. Minimum, maximum and average packet delay is calculated, together with chart showing packet delay history. Statistical data are updated automatically, when new packet is delivered to its final destination. For ICMP packets (ping), their final destination is the source that created these packets.

How to select simulation rule statistics

Basic simulation rule statistics are shown on upper left side, after simulation is started. It is possible to display detail statistics about each simulation rule by clicking on button "Show stats". This will open a panel on the bottom with all statistical data.
